安装jenkins

准备工作 安装java环境

  1. 配置yum源
  1. 导入公钥

  1. 安装jenkins

    yum install jenkins -y

  2. 修改用户

    vi /etc/sysconfig/jenkins

  3. 配置文件中加入java路径

    vim /etc/rc.d/init.d/jenkins

  4. 启动
    systemctl start jenkins

  5. 初次密码路径
    /var/lib/jenkins/secrets/initialAdminPassword
